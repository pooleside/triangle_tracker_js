# _Leap Year (JavaScript)_

##### _Create an app to determine if you have a triangle._

#### By _Leslie Poole_

## Description
Web page that lets the user input the lengths of the sides of a triangle, and returns whether they form an equilateral, isosceles, or scalene triangle.  If not, it will let the user know they do not have a triangle.  


## Setup

* Open the triangle-tracker.html file in your browser(Chrome) to run the program.

## Technologies Used

JavaScript, JQuery, Mocha, Chai, Bootstrap, HTML, and CSS
